Сервер
Працює на Node JS з використанням фреймворку Express JS
Для завантаження файлів використовується сервіс Cloudinary
Для запуску потрібно:
1. В папці server відкрити термінал і прописати команду npm install
2. Створити новий файл з назвою .env
3. Скопіювати вміст файлу .env.example в файл .env
4. Як PORT вказати наприклад 5000
5. Зареєструватись на сайті cloudinary.com, отримати CLOUD_NAME, API_KEY і API_SECRET, їх потрібно прописати в .env файл
6. В терміналі прописати команду npm run start:dev
7. Для підтримки завантаження архівів і pdf на cloudinary порібно зайти в налаштування профілю - Security і поставити в кінці галку біля Allow delivery of PDF and ZIP files.

Клієнт
Працює на реакті, створений через звичайний create-react-app
1. В папці client відкрити новий термінал і прописати команду npm install
2. Після закінчення виконання команди npm install прописати нову команду npm start, з часом відкриється нова вкладка в браузері
3. В файлі src/App.js прописано базову урл для підключення до сервера, якщо на сервері вказаний інший порт а не 5000 то потрібно змінити.
   Так само цю урл потрібно змінити при деплої на урл сервера з хостинга. В кінці обов'язково НЕ має бути /
